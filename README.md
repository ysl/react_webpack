# React + Webpack

## Init
```
npm install --dev
```

## Run in dev mode
```
npm run dev
```
Open http://localhost:8088, it support live reload.


## Run in production mode
```
npm run build
cd build
python -m SimpleHTTPServer 8088
```
Open http://localhost:8088
